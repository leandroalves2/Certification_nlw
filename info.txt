API REST
Método comuns
 - GET - Buscar uma informação
 - PUT - Alterar uma informação
 - POST - Inserir uma informação
 - DELETE - Remover uma informação
 - PATCH - Alterar PONTUAL de uma informação

 Os tipos de parâmetros
 - Body - { }
 - Query Params - http://localhost:8085/users?nome=Leandro&idade=29 OPCIONAIS
 - Parâmetros de rotas - http://localhost:8085/users/{nome}/{idade}
  
 ORM -> Object Relational Mapper

 Reposiótio (Repository) -> Interagir com o banco de dados.
